# ClassIn导航站项目说明

## 项目概述

ClassIn导航站是一个面向ClassIn用户的公共导航平台，名为"四季亭导航站"。该项目主要提供ClassIn群聊链接的分类导航服务，同时包含音乐播放器、网页跳转等功能。用户可以通过该网站快速访问不同的ClassIn群组和相关资源。

## 技术栈

- **前端技术**: HTML5, CSS3, JavaScript (ES6+)
- **样式实现**: 主要使用内联CSS样式，包含响应式设计
- **音乐播放器**: 独立的CSS和JS文件实现可折叠播放器
- **图标库**: Font Awesome 6.4.0
- **部署平台**: 基于GitHub Pages或类似Web服务器部署
- **域名**: classin.luoqing5203789.dpdns.org (通过CNAME文件配置)

## 功能特性

### 1. 主页功能
- 提供导航站介绍和功能跳转
- 集成音乐播放器控件（固定在浏览器右侧边缘）
- 公告展示区域
- 友情链接和雨云服务推广
- 响应式设计适配不同设备

### 2. 音乐播放器
- 固定在浏览器右侧的可折叠播放器（默认收起状态）
- 从GitHub仓库获取音乐文件列表
- 播放进度控制和时间显示
- 上一首/下一首切换功能
- 键盘快捷键支持（空格键播放/暂停，Ctrl+左/右箭头切换歌曲）
- 播放列表管理，支持自动循环播放
- 自适应移动端和桌面端显示

### 3. 导航分类系统
- 资源分类中心页面（center.html）
- ClassIn群组分类导航
- 网页跳转服务分类
- 模块化设计，易于扩展新分类

### 4. ClassIn群组导航
- 按组别展示ClassIn群组
- 每个组别下包含多个群组链接
- 直接跳转到ClassIn群组

### 5. 关于页面
- 开发者介绍页面
- 包含开发者头像和联系方式
- 团队成员信息展示

## 音乐播放器配置

音乐播放器从GitHub仓库获取音乐文件列表，需要配置以下参数：

```javascript
const musicConfig = {
    baseUrl: "https://luo202044.github.io/classinapi/",  // 替换为你的GitHub Pages音乐仓库地址
    apiFile: "api.txt",  // 包含音乐文件列表的文件名
    // ...
};
```

播放器会从 `baseUrl + apiFile` 路径获取音乐文件列表，该文件应包含每行一个音乐文件名。
播放器还包含模拟播放列表作为备用，以防GitHub API访问失败。

## 设计风格

- **主色调**: 粉色系（#FF66B2, #FF3385）作为主题色
- **辅助色**: 深粉色（#B3005E）、淡粉色（#FFE6F2）用于文本和背景
- **交互效果**: 包含悬停动画、渐变效果、阴影等现代UI元素
- **响应式设计**: 适配不同屏幕尺寸，包括移动设备
- **卡片式布局**: 使用卡片展示内容，视觉效果清晰

## 使用说明

1. 访问主页（index.html）查看导航站介绍和功能入口
2. 点击"点我跳转"按钮进入资源分类中心
3. 在资源分类中心选择ClassIn或其他分类
4. 在分类页面查看具体群组信息并点击跳转
5. 使用固定在右侧的音乐播放器享受音乐
6. 点击播放器左侧面板展开或收起播放器
7. 使用键盘快捷键控制播放（空格键播放/暂停，Ctrl+左右箭头切换歌曲）
